## Usage
1. Add JitPack to your project `build.gradle`

    - Kotlin DSL

    ```kotlin
    repositories {
    	...
    	maven(url = "https://jitpack.io")
    }
    ```

2. Add the dependency in the application `build.gradle (app)`
 
    - Kotlin DSL

    ```kotlin
    implementation("com.github.lock:android-common-ktx:<version>")
    ```
